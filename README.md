# 💰 Zorvyn – Finance Dashboard

## 🚀 Overview

SpendWise is a modern and responsive finance dashboard web application that helps users track income, manage expenses, and gain meaningful insights into their financial activities.

This project is built as part of a frontend developer internship assessment to demonstrate UI design, data handling, and interactive features.

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance, Income, and Expense summary cards
* Line chart showing balance trend over time
* Pie chart showing category-wise spending breakdown

### 📋 Transactions Management

* View all transactions with:

  * Date
  * Amount
  * Category
  * Type (Income/Expense)
* Features:

  * Search transactions
  * Filter by category/type
  * Sort by date and amount
  * Add, Edit, Delete transactions

### 👤 Role-Based UI

* Viewer:

  * Can only view data
* Admin:

  * Can add, edit, and delete transactions
* Role switch available in UI

### 🧠 Insights

* Highest spending category
* Monthly expense comparison
* Total savings calculation

### 💾 Data Persistence

* Data stored using localStorage
* Data remains after page refresh

### 📱 Responsive Design

* Fully responsive for mobile, tablet, and desktop

### 🎨 UI/UX

* Clean and modern design
* Smooth interactions and hover effects
* Handles empty states gracefully

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/coding-smit/Zorvyn-Financial-Dashboard.git
```

2. Navigate to project:

```
cd spendwise
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```
